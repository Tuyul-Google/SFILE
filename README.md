## Cara Menginstall
- **Pertama Jalankan Sekali Kode Dibawah Ini**
```
pkg update && pg upgrade
pkg install python2
pkg install python
pip2 install --upgrade pip
apt install git
```
- **Cara Menjalankan Tools** `SFILE`
```
git clone https://github.com/Kang-Ngulik/SFILE
cd SFILE
python2 -m pip install requirements.txt
python2 main.py
```
